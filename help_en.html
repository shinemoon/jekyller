üîß User Guide

First Time Use

1. Click the extension icon in the browser toolbar

2. Configure your GitHub information in settings:

  • GitHub username
  • Repository name
  • Access token
Tip: How to get a GitHub Token?
Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Generate new token
Make sure to check repo permissions.

Create New Post

1. Click the "New Post" button

2. Write your content in the editor

3. Fill in the Front Matter (title, date, categories, etc.)

4. Click the "Publish" button to upload to GitHub

Edit Existing Post

1. Select the post you want to edit from the list

2. Modify the content and save

3. Click the "Update" button to sync to GitHub

‚å®Ô∏è Key Binding Modes

The editor supports multiple key binding modes to suit different editing preferences:

üîπ Normal Mode (Default)

Standard editor mode with common keyboard shortcuts.

  • Ctrl+S - Save document
  • Ctrl+Z - Undo
  • Ctrl+Y / Ctrl+Shift+Z - Redo
  • Ctrl+F - Find
  • Ctrl+H - Replace
  • Ctrl+A - Select all
  • Ctrl+/ - Toggle comment
  • Home / End - Jump to line start/end
  • Ctrl+Home / Ctrl+End - Jump to document start/end

üîπ Vim Mode

Provides familiar modal editing experience for Vim users. Supports Normal, Insert, and Visual modes.

Basic Operations:

  • i - Enter insert mode (before cursor)
  • a - Enter insert mode (after cursor)
  • I - Insert at line start
  • A - Insert at line end
  • o - Open new line below and insert
  • O - Open new line above and insert
  • Esc - Return to Normal mode

Motion Commands:

  • h/j/k/l - Left/down/up/right
  • w/b - Forward/backward one word
  • 0 / ^ - Jump to line start
  • $ - Jump to line end
  • gg - Jump to document start
  • G - Jump to document end
  • {number}G - Jump to specific line
  • % - Jump to matching bracket

Edit Commands:

  • x - Delete character under cursor
  • dd - Delete current line
  • D - Delete to line end
  • yy - Yank (copy) current line
  • p / P - Paste after/before cursor
  • u - Undo
  • Ctrl+r - Redo
  • . - Repeat last operation

Visual Mode:

  • v - Enter character-wise visual mode
  • V - Enter line-wise visual mode
  • Ctrl+v - Enter block-wise visual mode
  • After selection: d(delete), y(yank), c(change), etc.

Ex Commands (type : in Normal mode):

  • :w - Write (save)
  • :q - Quit
  • :wq - Write and quit
  • :{number} - Jump to specific line
  • :s/old/new/g - Substitute in current line
  • :%s/old/new/g - Substitute in entire file
  • :h / :help - Show command help list

Extended Ex Commands (Editor-specific):

  • :layout / :l - Switch editor layout (full/single/preview)
  • :switch / :s - Switch theme (light/dark)
  • :new / :n - Create new post
  • :published / :pu - Mark current post as published
  • :unpublished / :un - Mark current post as unpublished
  • :language [en|zh] / :lang [en|zh] - Switch UI language (auto-toggle if no argument)
Vim Mode Tip:
Type :help or :h to see a list of all available Ex commands.

üîπ Sublime Mode

Emulates Sublime Text editor shortcuts, perfect for Sublime users.

  • Ctrl+D - Select next occurrence
  • Ctrl+K, Ctrl+D - Skip current selection, select next
  • Alt+F3 - Select all occurrences
  • Ctrl+L - Select current line
  • Ctrl+Shift+K - Delete line
  • Ctrl+Shift+D - Duplicate line
  • Ctrl+] / Ctrl+[ - Indent/outdent
  • Ctrl+Shift+Up/Down - Move line up/down
  • Ctrl+/ - Toggle line comment
  • Ctrl+Shift+/ - Toggle block comment
  • Ctrl+P - Quick file jump
  • Ctrl+G - Go to line

üîπ Emacs Mode

Key bindings for Emacs users.

  • Ctrl+F/B - Forward/backward character
  • Ctrl+N/P - Next/previous line
  • Ctrl+A/E - Beginning/end of line
  • Alt+F/B - Forward/backward word
  • Ctrl+K - Kill to end of line
  • Ctrl+D - Delete character
  • Ctrl+Y - Yank (paste)
  • Ctrl+Space - Set mark
  • Ctrl+W - Kill region
  • Alt+W - Copy region

Cannot connect to GitHub?
